# PWA Icons Required

The following icons are required for the PWA manifest.json to work properly:

## Required Icons (PNG format):
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Optional Icons:
- dashboard-icon.png (96x96) - For dashboard shortcut
- calculator-icon.png (96x96) - For calculator shortcut

## How to Generate:
1. Create a master icon at 512x512 pixels
2. Use a tool like https://realfavicongenerator.net/ to generate all sizes
3. Place all icons in this folder

## Icon Guidelines:
- Use PNG format with transparency
- Design should be simple and recognizable
- Use the ZYN Trade brand colors (#00d4ff cyan, #7c3aed purple)
